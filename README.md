### Usage ###
The user will be reqired to login if she/he already used the site. If the user is new, they will have to sign up.

Once you sign up/login, the user will be greeted by a dashboard where, the user will see their jobs and recent activity.

Profile contains the user data which the user can use.

Jobs is where the user can find a job and apply for it. It will be added to the dashboard.
